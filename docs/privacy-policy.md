# ShortcutBoard プライバシーポリシー

最終更新日: 2026-06-15

本プライバシーポリシーは、Windows 向けアプリ **ShortcutBoard**（以下「本アプリ」）における
利用者情報の取り扱いについて定めるものです。

## 1. 基本方針
本アプリは、利用者のプライバシーを最大限に尊重します。
**本アプリは個人情報を外部に送信せず、ネットワーク通信を一切行いません。**

## 2. 収集しない情報
本アプリは、以下を含むいかなる個人情報も**収集しません**。
- 氏名
- メールアドレス
- 位置情報
- 連絡先（アドレス帳など）
- 写真・カメラ・画像
- 音声・マイク
- 閲覧履歴・利用統計・解析データ
- 広告識別子

## 3. 本アプリが扱うデータ（ローカル保存のみ）
利用者が本アプリに登録・設定する以下のデータは、**利用者ご自身の PC 内にのみ保存されます**。
- 登録したショートカットの情報（キー操作・名前・説明・カテゴリー・並び順）
- アプリの設定（ホットキー、自動起動の有無、表示設定など）

これらのデータは次の場所に保存されます。

```
%AppData%\ShortcutBoard
```

（通常は `C:\Users\<ユーザー名>\AppData\Roaming\ShortcutBoard`。
Microsoft Store 版ではパッケージ専用の保存領域に保存される場合があります。）

## 4. データの利用目的
保存されるデータは、**本アプリの機能（ショートカット一覧の表示・編集・検索・並べ替え等）を
提供するためだけに使用されます**。それ以外の目的には使用しません。

## 5. 外部送信・第三者提供
- 本アプリはネットワーク通信を行わないため、データを外部サーバーへ送信しません。
- 利用者のデータを**第三者に販売・共有・提供することは一切ありません**。

## 6. データの保持と削除
- 本アプリをアンインストールしても、**登録データがローカルに残る場合があります**
  （再インストール時に引き継げるようにするため）。
- データを完全に削除したい場合は、以下のフォルダーを手動で削除してください。

```
%AppData%\ShortcutBoard
```

- フォルダーを削除すると、登録したショートカット・設定はすべて消去されます。
