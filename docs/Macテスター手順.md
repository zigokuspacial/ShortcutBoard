# ShortcutBoard（Mac版）テスター手順 — 最小版

Apple Silicon Mac で動作確認をお願いします。所要 5〜10 分です。

## 0. 受け取るもの
- `ShortcutBoard-AppleSilicon.dmg`（.NET などの事前インストールは不要）

## 1. インストール
1. `ShortcutBoard-AppleSilicon.dmg` をダブルクリックして開く
2. 中の **`ShortcutBoard.app`** を「アプリケーション」フォルダにドラッグ（その場起動でも可）

## 2. 初回起動（重要）
未署名アプリのため、普通のダブルクリックだと「開発元を確認できません」で止まります。
1. `ShortcutBoard.app` を **右クリック →「開く」** →「開く」を選択
2. 初回案内のウィンドウが出て、画面上部の **メニューバーに「S」アイコン** が表示されればOK

## 3. 確認チェックリスト
- [ ] **Ctrl + Shift + A** で一覧が画面中央に表示される
- [ ] ウィンドウの外側をクリック、または **Esc** で閉じる
- [ ] メニューバーの「S」をクリック →「一覧を開く」で表示される
- [ ] 右上 **「編集」** → 各行に ↑ ↓ 編集 削除 が出る
- [ ] **「＋ 追加」** → キー欄に「Ctrl/Alt/Cmd＋キー」を押すと取得、または直接入力 → 名前・カテゴリ入力 → **「保存」**
- [ ] **↑ / ↓** で並べ替え、**削除**（確認あり）
- [ ] アプリを再起動しても、追加・並び順が残っている
- [ ] メニューバー →「設定」→ ホットキー変更／「ログイン時に自動起動」を切替 → 反映される
- [ ] メニューバー →「終了」で終了できる

## 4. うまくいかないとき
- ログを共有してください：`~/Library/Application Support/ShortcutBoard/logs/`
- 特に確認したい挙動：
  - グローバルホットキー（他アプリと競合していないか。競合時は「設定」で別キーに変更できるか）
  - メニューバー常駐・外側クリックでの非表示

## 5. アンインストール
- `ShortcutBoard.app` をゴミ箱へ。
- 自動起動を有効にした場合は `~/Library/LaunchAgents/com.shortcutboard.app.plist` も削除。
- 登録データは `~/Library/Application Support/ShortcutBoard/` に残ります（不要なら手動削除）。

---
※ Intel Mac の場合は `ShortcutBoard-Intel.dmg` を使用してください（手順は同じ）。
