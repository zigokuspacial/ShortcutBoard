## 詳細説明（Description）
```
ShortcutBoard は、よく使うショートカットキーを「自分用の早見表」としてまとめておける、軽量な常駐アプリです。

「あのショートカット、なんだっけ？」を、キー1つで解決します。

■ 主な機能
・Ctrl + Shift + A を押すと、どのアプリを使っていても画面中央に一覧を表示
・項目の追加・編集・削除・並べ替えが簡単（右上「編集」から）
・キーワード検索・カテゴリ絞り込みで、たくさん登録してもすぐ見つかる
・一覧の項目を選んで Enter／ダブルクリックで、ショートカットキーをコピー
・通知領域（メニューバー）に常駐。Esc やウィンドウ外クリックで閉じる
・ダークなUIで見やすい
・ログイン時の自動起動に対応
・ホットキーは設定から自由に変更可能

■ こんな人に
・新しいアプリのショートカットをよく忘れる
・自分専用のショートカット一覧を手元に置いておきたい
・作業効率を上げたい

■ プライバシー・安全性
・ネットワーク通信を一切行いません
・登録したデータはご自身の PC 内にのみ保存されます
・収集・送信は行いません

初回起動時は、Windows標準の便利なショートカット10件がサンプルとして入っています。
そのまま自分用に書き換えてお使いください。
```
