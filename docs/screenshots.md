# スクリーンショット撮影ガイド（リリース用）

README やストアページ、SNS で使う画像の撮影リストです。
撮影した画像は `docs/images/` に保存してください（README は `docs/images/main.png` を参照）。

## 撮影前の準備
- 通知領域の「S」→「設定」でホットキーが `Ctrl + Shift + A` になっていることを確認
- 見栄えのため、サンプル/自分のショートカットが数件入った状態にする
- ダーク背景のデスクトップだと一覧の枠が映えます

## 撮りたい画面

| ファイル名 | 内容 | 撮り方 |
|---|---|---|
| `images/main.png` | **一覧（通常表示）** ＝ メイン画像 | `Ctrl + Shift + A` で一覧表示 → ウィンドウだけを撮影（Alt+PrintScreen でアクティブウィンドウのみ） |
| `images/edit.png` | **編集モード** | 一覧右上の「編集」を押し、各行に ↑↓・編集・削除が出た状態 |
| `images/add.png` | **項目の追加** | 編集モードで「＋ 追加」を押し、入力欄が開いた状態 |
| `images/search.png` | **検索・絞り込み** | 上部の検索欄にキーワードを入れて絞り込んだ状態（任意） |
| `images/tray.png` | **通知領域メニュー** | タスクバー右下の「S」を右クリックしてメニューを開いた状態（任意） |
| `images/onboarding.png` | **初回オンボーディング**（任意） | 新規環境での初回起動時に出るポップアップ |

## 撮影のコツ
- **Alt + PrintScreen**：最前面ウィンドウだけをクリップボードにコピー（余計な背景が入らない）
- **Win + Shift + S**：範囲を選んで撮影（このアプリにも登録されている定番ショートカット）
- GitHub の README で大きく見せるなら、メイン画像は横 800〜1000px 程度がきれい
- 個人情報（他アプリのウィンドウ、ファイル名など）が背景に写り込まないよう注意

## メイン画像が無い場合
README 冒頭の `![...](docs/images/main.png)` は、画像を置くまでリンク切れ表示になります。
画像を用意できないうちは、その行を一時的にコメントアウト/削除してもOKです。
