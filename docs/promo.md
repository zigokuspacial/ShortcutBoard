## 外部投稿に添える注意書き（安全性・コピペ用）

投稿やDLリンク紹介の際に、下のどちらかを添えると親切です（隠さず明記することで信頼感が上がります）。

**一文版：**
> ⚠️ 個人開発のため、初回起動時に Windows の警告（SmartScreen）が出る場合があります。ソースコード公開済み・通信なし・データはローカル保存です。

**詳しめ版：**
> ⚠️ コード署名を付けていない個人開発アプリのため、初回に「WindowsによってPCが保護されました」が出ることがあります（ウイルスではありません）。
> ・ソースコードは GitHub で公開しています
> ・ネットワーク通信は行いません
> ・登録データは自分の PC 内（%AppData%\ShortcutBoard）にのみ保存されます
> 不安な場合は、ソースコードと配布元（Releases）を確認してから実行してください。
