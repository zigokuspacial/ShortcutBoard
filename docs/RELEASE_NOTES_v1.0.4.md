## ShortcutBoard v1.0.4 🎉（初回リリース）

よく使うショートカットキーを自分用にまとめておける Windows 常駐アプリです。
**`Ctrl + Shift + A`** を押すだけで、いつでも一覧をパッと表示できます。

### ダウンロード
下の **`ShortcutBoard-Setup.exe`** をダウンロードして実行するだけ。
`.NET などの事前インストールは不要`・無料・**Windows 10 / 11（64bit）** 対応。

### インストール
1. `ShortcutBoard-Setup.exe` を実行
2. **「WindowsによってPCが保護されました」** と出たら → **「詳細情報」→「実行」**
   （個人作成で署名がないための警告です。動作に問題はありません）
3. ウィザードで「**Windows 起動時に自動起動する**」にチェック（おすすめ）→ 完了

### 使い方
- `Ctrl + Shift + A` … 一覧を表示
- `Esc` / ウィンドウ外クリック … 閉じる
- 右上「編集」 … 追加・編集・削除・並べ替え
- `Enter` / ダブルクリック … 選択中のキーをコピー
- 通知領域（タスクバー右下）の「S」… 右クリックで「一覧を開く / 設定 / 終了」

初回は Windows 標準の便利ショートカット10件がサンプルで入っています。
そのまま自分用に書き換えて使ってください。

### 主な機能
- グローバルホットキーで一覧表示（ホットキーは設定から変更可）
- 検索・カテゴリ絞り込み
- 項目の追加・編集・削除（確認ダイアログなしで即削除＋「元に戻す」）・並べ替え
- メニューバー常駐・ダークUI
- データはローカル保存（`%AppData%\ShortcutBoard`）。通信は一切行いません

### 注意事項
- コード署名なしのため、初回に SmartScreen 警告が出ます（「詳細情報」→「実行」で回避）
- Windows 専用です
- アンインストールしても登録データは残ります（再インストール時に引き継ぎ）
